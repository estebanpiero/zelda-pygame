# Zelda Pygame

A Zelda-style 2D pixel game using Python and Pygame.

## Setup

1. Install Python 3 and Pygame.

2. Clone the repository:

    git clone https://github.com/estebanpiero/zelda-pygame.git

3. Navigate to the project directory:

    cd zelda-pygame

4. Run the game:

    ## Controls

    - Arrow Keys: Move
    - Spacebar: Attack
